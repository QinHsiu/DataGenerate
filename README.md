### 文件结构,初始版本1.0
##### dataGenerate.csv:数据生成的保存文件
##### dataGenerate.py:负责生成数据的主要函数，包括架构
##### dataGenerateUI.py:会显示一个框架，负责将参数输入过程可视化
##### main.py:主要的运行函数
##### utils.py:包含所有套件函数，各种分布函数以及其他函数
